blender

Manual、

基本操作：

🖱️ 介面與控制

  • 旋轉視角：中鍵滑鼠拖曳

  • 平移視角：Shift + 中鍵拖曳

  • 縮放視角：滾輪滑動

🔧 模型操作

  • 選取物件：左鍵點擊

  • 移動：G（Grab）

  • 旋轉：R（Rotate）

  • 縮放：S（Scale）

  • 確認操作：左鍵點一下或 Enter

  • 取消操作：右鍵點一下或 Esc

🧱 模式切換

  • Object Mode：物件層級操作（預設）

  • Edit Mode：細部編輯（按 Tab 切換）

📏 快捷鍵小技巧

  • X：刪除物件

  • Shift + A：新增物件（Add）

  • Ctrl + Z / Shift + Ctrl + Z：復原 / 重做

  • NumPad 1/3/7：前視 / 側視 / 上視（搭配 Ctrl 反向）
